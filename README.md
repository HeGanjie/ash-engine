# ash-engine

3d game engine dev learning project.

### How to play
1. clone the project
2. `npm install`
3. `npm run dev`
4. access http://127.0.0.1:8080/

### Run result

Rotating cube:

[ash-engine-preview](https://codesandbox.io/s/github/HeGanjie/ash-engine)
